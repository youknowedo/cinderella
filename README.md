Reorder your Spotify playlists based on BPM. Turn your random ordered playlist into something that makes sense. 
Made with SvelteKit and the Spotify Web API

## How to use

1. Go to the [cinderella website](https://cinderella.sigfredo.dev)
2. Click *Select Playlist* and authenticate on Spotify's website if necessary
3. Select your playlist and click on *Drop the shoe*
4. Select how you want your playlist to be sorted
5. Click on *Update Playlist*
6. Enjoy!

### Self-host/Development

```bash
git clone https://github.com/youknowedo/cinderella.git
cd cinderella
bun i
bun dev
```
